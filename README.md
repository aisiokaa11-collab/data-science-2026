## Deskripsi Repository

Halo, saya Oka Maulana. Repository ini dibuat sebagai sarana untuk mendokumentasikan seluruh kegiatan pembelajaran pada mata kuliah Data Science. Melalui repository ini, saya menyimpan berbagai tugas, latihan, serta hasil praktikum yang dikerjakan selama proses perkuliahan sebagai bahan pembelajaran dan referensi di masa mendatang.

Isi repository ini terdiri dari beberapa notebook Jupyter yang disusun berdasarkan pertemuan kuliah. Setiap notebook membahas topik yang berbeda, mulai dari dasar-dasar pemrograman Python untuk pengolahan data, analisis statistik, visualisasi data, persiapan data, hingga penerapan konsep Machine Learning. Selain berisi kode program, notebook juga dilengkapi dengan penjelasan, hasil analisis, dan visualisasi yang membantu dalam memahami setiap materi yang dipelajari.

Dengan adanya repository ini, saya berharap dapat mendokumentasikan perkembangan kemampuan dalam bidang Data Science sekaligus membangun portofolio akademik yang dapat digunakan sebagai bukti proses belajar dan penguasaan berbagai teknik analisis data menggunakan Python dan library pendukung lainnya.
